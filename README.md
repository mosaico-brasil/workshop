# Análise de Dados Públicos
Com o projeto Base de Dados e o Google Cloud

Workshop ministrado durante a Feira de Ciências e Mostra Científica de Itaituba (FECMITA) nos dias 26, 27 e 28 de novembro.

A programação completa pode ser consultada [aqui](https://www.instagram.com/p/DRVO3MFkS7N/).

## Ementa

### Título

Análise e Visualização de Dados Públicos: Do BigQuery ao Looker

Carga Horária

4 horas

## Objetivos

### Objetivo Geral

Capacitar os participantes na análise e visualização de dados públicos brasileiros utilizando ferramentas de Big Data em nuvem, desde a consulta até a apresentação de insights através de dashboards interativos com o Looker.

### Objetivos Específicos

- Compreender a estrutura e o funcionamento da plataforma Base dos Dados (https://basedosdados.org)
- Navegar e explorar conjuntos de dados no _datalake_ do BigQuery
- Desenvolver consultas em SQL para extração e análise de dados
- Criar visualizações e dashboards no Google Looker para comunicação de resultados

## Conteúdo Programático

1. Introdução ao projeto Base dos Dados
   - Apresentação do projeto e sua missão
   - Exploração do catálogo de dados disponíveis
   - Casos de uso e aplicações práticas

2. Fundamentos do BigQuery
   - Conceitos de data lake e armazenamento em nuvem
   - Interface e navegação no BigQuery
   - Estrutura de tabelas e metadados

3. SQL para Análise de Dados
   - Consultas básicas: SELECT, WHERE, ORDER BY
   - Agregações e funções: COUNT, SUM, AVG, GROUP BY
   - Joins e relacionamento entre tabelas
   - Filtragem e refinamento de dados

4. Visualização com Google Looker
   - Conexão do Looker com BigQuery
   - Criação de gráficos e visualizações
   - Desenvolvimento de dashboards interativos
   - Boas práticas de apresentação de dados

## Metodologia

Workshop prático com atividades hands-on. Os participantes realizarão exercícios guiados para consolidar o aprendizado, desde a consulta de dados até a criação de visualizações completas.

Pré-requisitos

- Conta Google (Gmail) para acesso ao BigQuery e Looker
- Computador do Laboratório ou Notebook pessoal com acesso à internet
- Não é necessário conhecimento prévio em SQL ou análise de dados

## Público-Alvo

Estudantes dos cursos técnicos, de graduação e pós-graduação interessados em análise de dados, ciência de dados, pesquisa acadêmica e gestão de informações públicas.
